Tech Quiz Application

Overview

This is a simple interactive quiz application built using HTML, CSS, and Vanilla JavaScript. 
The application is designed to test users with a set of timed multiple-choice questions and provide instant scoring feedback.

Features

Contains 10 multiple-choice questions,
Each question has a 10-second countdown timer,
The timer visually changes from blue to red when it reaches 3 seconds remaining,
Automatic progression to the next question when time expires,
Immediate feedback on selected answers,
Correct answers are highlighted when a wrong answer is selected or time runs out,
Score tracking system where each correct answer awards 10 points.

How It Works

The quiz starts immediately when the application is launched,
A question is displayed with four answer options,
The user has 10 seconds to select an answer,
If the selected answer is correct, 10 points are added to the score,
If the answer is incorrect or time expires, the correct answer is shown,
The quiz continues until all 10 questions are completed,
The final score is displayed at the end of the quiz

Scoring System

Correct answer: +10 points
Incorrect answer or timeout: 0 points


Technical Stack
HTML for structure
CSS for styling and timer animations
JavaScript for logic, timing, and score handling

Project Structure

index.html – Main HTML file
style.css – Styling and layout
script.js – Quiz logic and functionality
Usage

Clone the repository:
git clone https://github.com/dev-surds/Tech-quize.git

Open the project folder and run the index.html file in any modern web browser.
